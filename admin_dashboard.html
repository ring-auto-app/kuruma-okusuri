管理者ダッシュボード

管理者ゲート

Script Properties の ADMIN_BACKDOOR_LOGIN_ID / ADMIN_BACKDOOR_PASSWORD_RECORD のみに保存した専用ログインです（リポジトリには含めません）。

通常のユーザーログインはこちら（シート上で B 列が admin のアカウントでも入れます）

即時監視（本日・JST）

System_Logs から当日分を集計。事故検知を最優先に表示します。

お知らせ一斉メール配信

登録メールアドレス（一般ユーザー: users_v1 N列 / 工場・事業者: shops L列）へ一斉送信します。誤送信防止のため、実行前に対象件数の確認を行います。

システム設定

スプレッドシートの「config」シートに保存されます。保存するとサーバー側の挙動（車検リマインドの日数など）に反映されます。
