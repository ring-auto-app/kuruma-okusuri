# icons フォルダ

PWA のアイコンと、スプラッシュ画面用ロゴを置きます。

**注意:** 画像は `README.md` の中ではなく、この **`icons` フォルダ（エクスプローラー上の中身）** に置きます。ファイル名は **`splash-logo.png`** にすると [splash.html](../splash.html) と一致します（`IMG_7990.png` など別名の場合はリネームするか、`splash.html` の `src` をその名前に合わせてください）。

## ファイル一覧

| ファイル | 用途 |
|----------|------|
| `icon-192.png` | マニフェスト・ホーム画面（192×192 推奨） |
| `icon-512.png` | マニフェスト（512×512 推奨） |
| `splash-logo.png` | **起動時スプラッシュ**（透明 PNG 可。長辺 512〜1024px 程度で十分） |

## スプラッシュロゴの入れ方

1. 画像編集ソフトでロゴを **PNG（背景透明のまま）** で書き出す。
2. ファイル名を **`splash-logo.png`** にする（名前を変えた場合は [splash.html](../splash.html) の `<img src="...">` を同じ名前に合わせる）。
3. この **`icons` フォルダにドラッグ＆ドロップ**で上書き保存する。
4. ブラウザで `splash.html` を開き直す（強制再読み込み: `Ctrl+Shift+R`）。

大きすぎる原寸（3000px 級）でも動きますが、容量のため **長辺 768〜1024px** に縮小したコピーを置くと読み込みが速いです。

## アプリアイコン（192 / 512）の入れ方

1. 正方形 **192×192** → `icon-192.png` に上書き。
2. 正方形 **512×512** → `icon-512.png` に上書き。
3. マスク対応のため、ロゴ周りに **十分な余白** を付けるとホーム画面で切れにくいです。

参照: [manifest.json](../manifest.json)・[service-worker.js](../service-worker.js)
